**To-Do List**

Aplicação de lista de tarefas desenvolvida para praticar lógica de programação e manipulação do DOM com JavaScript.

**Funcionalidades**

- Adicionar tarefas  
- Marcar como concluídas  
- Interface clean e intuitiva  
- Armazenamento no navegador  

**Tecnologias**

- HTML  
- CSS  
- JavaScript
